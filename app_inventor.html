Purpose of the App


The purpose of our app is to give people the ability to make a list at will. Our app can be used to list whatever the user desires. For example, you can make 
a grocery list or a list of school supplies. You can also use it to keep track of homework assignments. Key features of the app include the alphabetize feature which rearranges your 
list in alphabetical order when clicked. Another key feature is the suggestion button that when clicked a random object from the list pops up. You can be creative and give the suggestion 
button different purposes. The final key feature is the calculator. The calculator is a bonus feature and allows the user to add and subtract whole numbers.
